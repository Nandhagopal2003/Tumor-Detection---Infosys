About Us

The Iris Tumour Detection project aims to build an AI-based system that helps in the early detection of iris tumors using Convolutional Neural Networks (CNN). The project involves processing medical images, training a CNN model for accurate tumor detection, and developing a user-friendly interface for users to interact with the system.

Key components of the project include:

  • Image Processing: Preprocessing medical images by resizing, normalizing, and augmenting data to improve model accuracy.
  • CNN Model: Designing, training, and evaluating a CNN to identify potential iris tumors in images.
  • Web Interface: A frontend that allows users to easily upload images, view results, and interact with the system.

  • Registration Page and Login PageThe Registration Page and Login Page are integral parts of the user management system. The Registration Page allows new users to create accounts by providing their username and password, ensuring secure access. The Login Page lets users log in using their credentials to access the system's tumor detection features.

    • Together, these elements create a comprehensive and secure platform for users to diagnose iris tumors efficiently.
